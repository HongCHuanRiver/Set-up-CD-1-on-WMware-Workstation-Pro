# DC-1 靶场搭建笔记
本仓库记录了在VMware Wrokstation Pro 中搭建DC-1渗透测试靶场的完整流程，用于网安学习与实验
## 项目结构
- ‘搭建DC-1至WMware Workstation Pro.md’:核心操作文档，包括下载DC-1 zip 镜像文件，解压DC-1 zip 镜像文件，把DC-1.ova导入WMware Workstation Pro，在v mware workstation Pro里面配置DC-1等详细步骤
- ‘.gitignore’：Git忽略配置，排除繁琐的json文件
- ‘.gitattributes’:Git文本格式配置
## 使用方法
1. 确保有一个找得到的，不含中文路径，储存在15G以上的存储空间
2. 确保本地安装WMware workstation Pro 15+版本   
3. 打开‘搭建DC-1至WMware Workstation Pro.md’文件
4. 按照‘搭建DC-1至WMware Workstation Pro.md’文件的说明完成靶场搭建(内置镜像文件下载地址 )
## 注意事项 
- 本项目仅用于合法的网络安全学习与实验 
- 请勿在未授权环境中使用相关技术
